# The Lab Is a Lie

Most robotic systems look smart  
because the environment is cooperating.

Good lighting.  
Silence.  
Clear inputs.

Reality doesn’t behave like that.

If a system only works when conditions are perfect,  
it doesn’t really work.

For Nomi, the happy path is easy.  
Clean visuals.  
No background noise.  
User facing the camera.

That’s not where it lives.

It lives in homes.

TVs running.  
Lights changing.  
Rooms cluttered.  
Sound bouncing off everything.

So we define “functional” using worst-case environments,  
not best-case ones.

If a companion fails when the room is dark  
or when a vacuum cleaner is running,  
it stops feeling like a companion.  
It starts feeling like a fragile gadget.

Robustness isn’t something you add later.  
It’s the baseline.

We don’t build for controlled conditions.  
We build for the world as it is.
